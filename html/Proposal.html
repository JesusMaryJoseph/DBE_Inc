  2. In 1975 MOS Technology introduced the 6502 8-bit microprocessor.

  3. In 1977, based on these two technologies, Companies like Apple, Radio Shack, Commodore introduced the first Personal Computers. The magazine Byte introduced them as:

  1. Form a Company that will:

            1. This is what they call an "IDE"(Integrated Development Environment) software.

          4. "GitHub":
            1. GitHub is a "Version Control System" and "Collaboration Tool" used by Software Developers.

            4. When the MediaArts Manager wants to add images, videos or text to this website, it will be done in Visual Studio Code, which is setup to transfer data to and from their website on GitHub. They may only want to change the color of something, but even that must be done in a similar fashion.
          5. "Arduino IDE":
            1. This is only used in the Computer Programming Dept.
            2. It will be used for programming the three EEPROMs (one in the "Output Module" and two in the "Control Module").
            3. It will serve as an introduction to the "C/C++" programming language.
            4. Ben Eater's videos on YouTube will serve as a guide. No one will be expected to do this on their own.
            5. Note: This project is not well suited for learning any programming language, whether it be C/C++, JavaScript/HTML/CSS or anyother language. There is just too much to do outside that.
            6. It can, however, serve as a good introduction. I would be willing to help anyone on a specific thing they want to do on the Company Website.
            7. It is possible that I could add some specific material on the "Training" dropdown on Education Dept Page.
        4. Media Arts Software:
          1. GIMP:
            1. This is a free version of "Photoshop".
            2. Those working in the Media Arts Dept will be expected to take pictures through out this project as well as modifing them in GIMP.
            3. Many of these images should find their way to the various Depts on the website!
          2. DaVinciResolve or Kdenlive:
            1. DaVinciResolve & Kdenlive are video editing software.
            2. DaVinciResolve is a professional level program like Adobe's Premiere Pro. They have a free version for DaVinciResolve; however, it demands a fairly powerful computer, which, at present, is not available to us. Kdenlive will be the alternant option.
            3. Videos of all phases of this project should be taken and modified. Ben Eater's videos online should be used as examples.
          3. Inkscape:
            1. Inkscape is a free version of Adobe's Illustrator.
            2. Like GIMP it works with images but creates vector images rather than pixel-based images.
            3. The images of the Logic Gates were created in Inkscape. Note the difference file formats: .jpg, .png for GIMP images and .svg for Inkscape images!
            4. Learning both is highly recommended!
          4. Audacity:
            1. Audacity is a free version of Adobe's Audition. Both are Audio Editing Programs
            2. It probably has limited use in this project but if you do find you want to edit any audio then Audacity is available.
