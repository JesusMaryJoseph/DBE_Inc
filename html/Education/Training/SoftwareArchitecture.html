The RAM for this 8-bit Computer is composed of two 74LS169 chips, where each 74LS169 has a 4-bit address line ie (16 = 24 combinations). Consider that each each 74LS169 only has 64 bits of memory. The INPUTS(D1,D2, D3,D4) and OUTPUTS (O1,O2, O3,O4) are only 4-bits! (64bits/16). That is why two 74LS169s are needed- one 74LS169 for the 4 LeastSignificantBits(LSB) and one 74LS169 for the 4 MostSignificantBits(MSB).

The next 4 commands are needed to perform Addition(+) and Subtraction(+) in the ALU.

  1. LOAD from RAM into the 'A' Register of the ALU"!
  2. LOAD from RAM into the 'B' Register of the ALU" and ADD(+)!
  3. LOAD from RAM into the 'B' Register of the ALU" and SUBTRACT(-)!
  4. LOAD this # immediately into the 'A' Register

Instruction Set Architecture (ISA)
# English Mnemonic Binary
0 Do Nothing NOP 0000
1 LOAD from RAM into the ALU 'A' Register LDA 0001
2 LOAD from RAM into the ALU 'B' Register & Add ADD 0010
3 LOAD from RAM into the ALU 'B' Register & Subtract SUB 0011
4 STORE the contents of the 'A' Register into RAM! STA 0100
5 LOAD a # immediately into the 'A' Register! LDI 0101

NOTE: Command's 1,2,3 & 4 will require the 4-bit RAM address of the number to be added to the Command's Binary number. eg 0010 1010 is the instruction to "Load the number in RAM address 1010 into the ALU 'B' Register" which is immediately ADDED to the contents of the ALU 'A' Register. 0100 1010 will STORE the contents of the 'A' Register into RAM at address 1010.

NOTE: Command 5 requires the "NUMBER(0-15)" to be added to the ALU 'A' Register. eg 0101 0110 is the 8-bit instruction to load the Number 3 (0110) immediately into the 'A' Register.
